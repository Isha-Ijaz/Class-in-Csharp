 Class in C#

This project has simple notes and examples about Class in C#.

Topics covered:
1. What is a class and an object
2. How to create objects
3. Using methods inside classes
4. Creating and using multiple objects

Examples included:
1. Example1 shows a basic class and object (Person)
2. Example2 shows multiple objects of a class (Car)
3. Example3 shows a class with a simple method (Calculator)

Folder structure:
theory contains explanation of class and object
 examples contains C# code examples
